问题 解决方法
multiple definition 编译错误 使用 Redis 6.x + RedisBloom 2.2.14 组合，或添加 -fcommon 编译标志
ERR unknown command 'BF.ADD' 模块未加载，检查 loadmodule 路径是否正确，重启 Redis 后重试
