InnoDB Buffer Pool 详解1. 什么是 Buffer Pool？Buffer Pool 是 InnoDB 存储引擎 中一块内存区域，用于缓存数据页和索引页。它是 MySQL 中最重要的内存组件，直接影响数据库的读写性能。 简单理解：Buffer Pool 是 MySQL 的“内存数据缓存层”。 2. Buffer Pool 的作用核心作用 加速读操作：查询数据时，先从 Buffer Pool 中查找，若存在（命中）则直接返回，避免磁盘 I/O。 加速写操作：修改数据时，先修改 Buffer Pool 中的页，然后由后台线程异步刷回磁盘（Checkpoint 机制）。 带来的好处 功能 说明 减少磁盘 I/O 内存操作比磁盘快 1000+ 倍 提高并发能力 多线程共享 Buffer Pool 实现数据持久性 配合 Redo Log 保证事务不丢失 3.
